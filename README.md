# Sensiren GPT & Voice Sandbox

Welcome to the Sensiren GPT project! This repository contains a suite of web-based tools designed to facilitate interactive AI chat experiences and advanced voice experimentation.

## Components

### 1. Sensiren GPT (`sensirenGPT.html`)
A streamlined, responsive chat interface designed for seamless interaction with GPT-style large language models. This UI focuses on readability and user experience in conversational AI.

### 2. Voice Sandbox (`voice_sandbox.html`)
A dedicated testing environment for audio-centric features. It allows developers and users to experiment with:
- Web Speech API integration
- Text-to-Speech (TTS) synthesis
- Voice command recognition and processing
- Real-time audio feedback loops

## Getting Started

This project is built using standard web technologies (HTML/JavaScript) and does not require complex installation.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/sensiren-gpt-voice-sandbox.git
    ```
2.  **Open the application:**
    - Simply double-click `sensirenGPT.html` to launch the AI interface in your default browser.
    - Open `voice_sandbox.html` to explore the audio and voice experimentation features.

## Usage

- **AI Chat:** Use `sensirenGPT.html` to prototype chat prompts or integrate with an API backend for real-time responses.
- **Voice Testing:** Use `voice_sandbox.html` to verify microphone permissions and test the accuracy of speech synthesis across different browser engines.
